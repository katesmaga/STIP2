Відповіді на питання

1. Що таке HTTP?
HTTP (HyperText Transfer Protocol) – це протокол передачі даних у мережі для взаємодії клієнта і сервера.

  • GET – отримання даних
  • POST – надсилання нових даних
  • PUT – оновлення даних
  • DELETE – видалення даних

2. Заголовки HTTP
Передають додаткову інформацію між клієнтом і сервером.

  • User-Agent – інформація про браузер/пристрій
  • Content-Type – тип переданих даних
  • Authorization – автентифікація

3. Коди стану HTTP

  • 1xx – Інформаційні
  • 2xx – Успішні
  • 3xx – Перенаправлення
  • 4xx – Помилки клієнта
  • 5xx – Помилки сервера

4. REST API
REST API (Representational State Transfer) – стиль архітектури веб-сервісів.

  • Клієнт-сервер
  • Безстанова взаємодія
  • Кешування
  • Уніфікований інтерфейс
  • Шарованість

5. HTTPS
HTTPS використовує шифрування для безпеки даних.

  • Конфіденційність
  • Цілісність
  • Автентифікація
